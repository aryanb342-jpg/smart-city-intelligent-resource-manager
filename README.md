# EcoNexus – Smart City Intelligent Resource Manager 🌿

A full-stack intelligent platform for monitoring, analyzing, and optimizing urban resource consumption through real-time dashboards and predictive analytics.

---

## 🚀 Features

* ⚡ Electricity monitoring
* 💧 Water tracking
* 🌐 WiFi bandwidth analytics
* ♻️ Waste monitoring
* 📈 Historical data visualization
* 🤖 Predictive resource intelligence

---

## 🛠️ Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB
* Python
